# Biconditional Statements
**P => Q is not the same as Q => P**

Biconditional statements P <=> Q
ex:
(a is even) <=> (a is divisible by 2)

**Truth Table for Bionditionals**
| P | Q | P<=>Q |
|---|---|-----|
| T | T | T   |
| T | F | F  |
| F | T | F   |
| F | F | T   |
